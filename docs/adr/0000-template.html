How to set status (author reference)

On <main>, set one attribute data-adr-weight to the current milestone (−1 … 7). The page derives Current status and the Status log from that number. Full policy: ADR 0018.

Milestone scale (single order 0 → 7)
Value Meaning (current milestone)
-1 Status not set (or unknown)
0 Decision — Proposed
1 Decision — Accepted
2 Decision — Superseded
3 Documentation — Draft
4 Documentation — Ready
5 Implementation — Not started
6 Implementation — In progress
7 Implementation — Done

Example: data-adr-weight="4" → Current status is Documentation · Ready. Invalid values are clamped in docs/assets/docs-nav.js.
