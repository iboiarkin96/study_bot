Error log

POST /api/v1/errors

Store a learning mistake message. Optional conspectus_uuid links the entry to a conspectus owned by the same user; if set, the service verifies ownership via ConspectusRepository.get_for_client. Planned operationId: createErrorLog (implementation TBD).

Planned operationId: createErrorLog

Body and idempotency

Body: system_user_id, system_uuid, message; optional conspectus_uuid.

Idempotency: required. endpoint_path = "/api/v1/errors" (constant ERRORS_HTTP_BASE_PATH).

Success: 201 Created — ErrorLogResponse.
