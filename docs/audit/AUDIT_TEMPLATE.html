Assessment template

How to use this template

Copy this file to the correct subdirectory, rename it, and replace every placeholder. Do not change the section order or heading id attributes — they are referenced from the hub index and sidebar.

File location and naming

  • Folder: docs/audit/docs/ for documentation / DX / UI-UX assessments; docs/audit/api/ for REST / HTTP API assessments. Do not put new assessments in docs/audit/ root.
  • File name: YYYY-MM-DD-topic-assessment.html — ISO date first, kebab-case topic, then -assessment.html.
    Example: 2026-04-23-ui-ux-assessment.html.
  • Asset paths: files inside docs/audit/docs/ or docs/audit/api/ must use ../../assets/… (two levels up). Replace ../assets/… used in this template file.
  • Title alignment: put the same formation date in <title>, <h1>, Overview, and the first Page history row.
  • After creating: add a row to the matching stream index (docs/audit/docs/README.html or docs/audit/api/README.html) and update its Page history.
