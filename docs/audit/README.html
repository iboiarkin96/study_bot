Architecture & quality assessments

Overview

Maturity assessments for this repo. Each report lists reference practices (Table 1), scores how we match them (Table 2) with a short justification per row, then scoring arithmetic, Top-5 gaps with workflow status, and next steps. ADRs record decisions; assessments record how we compare to common practice on a given date.

For the template, file-naming rules, required HTML structure, and the copy-paste LLM prompt, see the Assessment template.
