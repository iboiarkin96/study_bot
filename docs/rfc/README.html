RFC Index

RFCs spell out how we build and run things: implementation detail and operational steps. ADRs record what we decided and why. Each numbered RFC sets data-rfc-weight on <main> (−1…7) so Current status and the Status log use the same lifecycle as ADRs (ADR 0018).
