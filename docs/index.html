Study App — Documentation

HTML documentation for the Study App API: architecture, engineering practices, developer guides, ADRs, runbooks, and generated Python API docs under docs/api/. When this tree is published (for example with GitHub Pages from the docs/ folder), this page is the site entry point. Changes to this documentation set are summarized in docs/CHANGELOG.md (separate from the root CHANGELOG.md, which tracks product and API changes).

GitHub Pages URL shape

If Pages uses Deploy from branch → /docs, the docs/ folder becomes the site root. Do not include /docs/ in the public URL path.

Wrong: …github.io/<repo>/docs/system-analysis.html
Right: …github.io/<repo>/system-analysis.html
